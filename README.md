# PU CSE Hub

Prime University — Department of CSE
69th / 262th Batch Portal

A modern student portal for managing class routines, notices, resources, group links, and important academic information for Prime University CSE students.

---

## Features

### Current Features

* Group links page
* Weekly class routine
* Important notices
* Course resources section
* Mobile responsive design
* Dark mode UI

### Planned Features

* Attendance calculator
* GPA calculator
* Teacher directory
* Admin dashboard
* Firebase authentication
* Push notifications
* File upload system

# Deployment Guide

## 1. Create Repository

Repository Name:

```text
pu-cse-hub
```

## 2. Upload Files

Push all project files to GitHub.

## 3. Enable GitHub Pages

Go to:

```text
Repository Settings → Pages
```

Then:

* Source → Deploy from branch
* Branch → main
* Folder → /root

Save.
